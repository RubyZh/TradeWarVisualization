Immersive Visualization Analysis Framework for News Content

Development Environment and Framework Overview


This framework is developed using the Unity 2021 engine, with the Meta Quest 3 virtual reality headset as the debugging device, to build a visualization and interaction system for immersive analysis of multi-source news text data. Compared to traditional two-dimensional web-based visualization methods, we leverage the spatial and immersive qualities of VR to design a “news information studio” environment suitable for quickly overviewing and filtering large-scale news data.

The system supports importing CSV-formatted data after converting it into structured JSON files (according to data item requirements), and dynamically renders modules such as timelines, country panels, and sentiment analysis panels to help users efficiently organize the timeline of news events under a specified theme, extract sentiment information, and generate semantic summaries in virtual reality, thereby enhancing their ability to understand hot topics both holistically and in detail.

Framework Advantage

Spatial Expression of Complex Information

Traditional 2D charts are limited in screen space, making it hard to present multidimensional news and emotion data clearly. This framework leverages VR’s spatial layout to distribute panels, timelines, and emotion charts across a 3D scene, enabling users to explore information through ray interaction and movement.

Efficient Filtering and Summary Matching

Users can filter news by time, country, and emotion to quickly locate key events. This enables smooth navigation from event-level overviews to text-level summaries, avoiding aimless content searching and improving preview efficiency.

Immersive Preview + 2D Deep Dive

By connecting the VR interface with deployed HTML chart systems (via 3D WebView), users gain spatial context first, then switch to detailed chart analysis, forming a closed-loop exploration between VR and web platforms.

Future Optimization Directions

Web Visualization Embedding and Interaction Integration

In the existing framework, we have imported the 3D Webview for Android toolkit. In the future, we will further support integration with previously built HTML websites to achieve the embedding and dynamic display of two-dimensional web visualization results in immersive scenarios. Furthermore, users can interact with country panels and news data cards in the 3D visualization space through information anchors or pop-up windows, enhancing information retrieval efficiency and cross-modal understanding capabilities.

Country Region Selection and Joint Analysis

The system currently supports independent analysis of individual country news panels. Moving forward, we will leverage the ray interaction mechanism in VR environments to enable users to select multiple country analysis panels via region selection, thereby constructing a cross-national joint analysis perspective. For example, after selecting a country, the system will automatically aggregate news data from multiple countries and dynamically generate interactive analysis views such as parallel coordinate charts (similar to the IATK toolkit, where users can dynamically capture coordinate axes for splicing and analysis), enhancing the spatial expression capabilities of multidimensional data. This enables users to intuitively compare differences and similarities between countries in terms of sentiment trends, source characteristics, and other aspects within an immersive environment, supporting cross-disciplinary exploration for international news media.

Integration of Open-Source Immersive Visualization Toolkits

We have successfully integrated the open-source immersive visualization toolkit IATK into the existing framework and resolved compatibility issues with certain versions. Future integration with its interesting and mature toolkit will enrich the visual forms and content of immersive visualization analysis.
