Data Import and Time Filtering

Introduction

This feature is designed to help users import news data through simple operations and filter and display data based on time ranges.
The data import process requires users to standardize news data into a specified format. For example, CSV files must be converted to JSON.
Sample scripts can be found at:

https://github.com/RubyZh/TradeWarVisualization/tree/main/preprocessing/unity_changeCSVtoJSON.py

Users can simply drag a JSON file into the demo panel and run the project. The system will automatically filter out invalid data and display the total number of news items.

The time filtering section uses two sliders to control the start and end times. Users can directly select the desired date range in the UI, and the system will automatically filter and display the news data within that time period, providing immediate feedback.

Core Design

NewsEntry class

Used to represent the structure of each news item, including basic information such as title, author, publication time, content, sentiment analysis, and several keywords (Key1, Key2, ...). These attributes serve as the basis for retrieval and classification during the analysis process.

NewsEntryList class

Used to store multiple news entries (NewsEntry), it serves as a container for all news records and supports loading and parsing all entry data from a JSON file. This feature aims to help users import news data through simple operations and filter and display data based on time ranges.

NewsDataLoader class

Responsible for loading, filtering, and storing data. It reads JSON-formatted news data via the LoadJSON method and parses it into a data collection of type NewsEntryList.

UI Notes

Time Filter Sliders

startDateSlider and endDateSlider are used to select the time range for news data, supporting precise filtering by day. The minimum and maximum values of the sliders are dynamically generated based on the valid date set availableDates.

Filter Status Display

filterStatusText is used to display the number of filtered news items in real time, helping users understand the results of the data filtering.

News Loading and Status

statusText displays the current loading status information, such as “Loading,” “Loading Complete,” or “Data Missing” prompts.
